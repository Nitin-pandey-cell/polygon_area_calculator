
# Polygon Area Calculator

## Overview

Polygon Area Calculator is an interactive JavaFX application that allows users to transform a square into any valid 3-sided or 4-sided polygon by dragging its vertices. The application dynamically computes and displays the polygon's area, side lengths, and internal angles in real time.

## Features

* Drag-and-drop vertex manipulation
* Real-time area calculation
* Dynamic side length measurement
* Automatic angle computation
* Interactive JavaFX-based GUI
* Support for valid triangles and quadrilaterals

## Tech Stack

* Java
* JavaFX

## Highlights

* Built an intuitive graphical interface for geometric visualization.
* Implemented mathematical algorithms for area, angle, and distance calculations.
* Delivered instant feedback through real-time updates as vertices are repositioned.

## Usage

1. Launch the application.
2. Drag any vertex to reshape the polygon.
3. Observe real-time updates of:

   * Polygon Area
   * Side Lengths
   * Internal Angles

The project serves as a practical demonstration of computational geometry concepts combined with interactive GUI development.
